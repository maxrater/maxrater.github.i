Ruby Loops vs. JavaScript Loops

JavaScript

Looping over object properties

        
          for (var property in obj) {
            console.log("o." + property + " = " + obj[property]);
          }

Regular For Loop with index

        

          for (var i = 0; i< 9; i++) {
            console.log("the index is" + i);
            // more statements
          }

While Loop

        

          var n = 0;
          var x = 0;

          while (n < 3) {
            n++;
            x += n;
          }

Ruby Loops

Example(Each Loop) 

          
            new_music = ["beyonce", "diplo", "bassnectar", "drake", "rihanna", "treasure fingers"]

            new_music.each do |music|
              p "#{music}"
            end

Example (For Loop)

          
          for num in 1...100
              puts num
          end

Example(While Loop)

          
            Fav_Movie = ["Drive", "Pulp Fiction", "American Sniper", "Interstellar", "John Wick"]
