Arrays

Arrays and Hashes are used in order to store information, and they each have their own way of doing it. Array's can be used to store a list of values in a single variable. When creating your array, keep in mind that you can code in any variant of numbers, you can repeat numbers, and nothing has to be in any numeric mandate. If we don’t want to use numbers or indices, we can use a new array structure called a hash!

A hash is a collection of data where each element of data is addressed by a name. Hashes are used when we don’t really care what order they are in. Hashes can be compared to JavaScript objects or Python dictionaries. A hash is an assembly of key-value pairs. A Hash syntax looks like this: *picture. Keys are on the left side and contain an arrow (=>), which is used to assign its value on the right side. This => is called a rocket.

A Hash can also be created by writting Hash.New and using bracket notation to list the keys and it's values. This creates a new empty hash and is equivalent to setting a variable equal to empty curly braces. A Hash.new uses bracket notation. We can access values in a hash just like an array.
